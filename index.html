Architecture

A four-agent pipeline

Agent 01

Web Search

Retrieves live headlines and analyst opinions using Claude's web search tool. This is the RAG retrieval step.

Agent 02

Sentiment

Analyzes investor sentiment grounded in the retrieved news data — not training knowledge.

Agent 03

Fundamentals

Evaluates business strength using real-time price data from Yahoo Finance alongside news context.

Agent 04

Synthesis

Combines all agent outputs into a final Buy / Hold / Sell verdict with confidence rating.

Features

Built for the Claude API

🔍

Retrieval-Augmented Generation

Agent 1 retrieves real web data before analysis begins, grounding all downstream reasoning in current facts.

🤝

Multi-agent collaboration

Four specialized agents run in sequence, each passing structured context to the next in the chain.

💬

Conversation memory

Ask follow-up questions after the report. The system maintains full context across every turn.

📈

Live price data

Real-time price and daily change pulled from Yahoo Finance and fed directly to the fundamentals agent.

🔢

Token tracking

Every session displays exact token consumption — ~3,000–6,000 tokens per full analysis.

⚡

No backend required

Runs entirely in the browser. The Claude API handles all computation — zero server infrastructure.
