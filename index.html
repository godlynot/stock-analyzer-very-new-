Architecture

A six-agent pipeline

Agent 01

Web Search

Retrieves live headlines and analyst opinions. The RAG retrieval step — grounding all downstream agents in current facts.

Agent 02

Sentiment

Scores investor sentiment based solely on the retrieved news data. Outputs a structured bullish or bearish assessment.

Agent 03

Fundamentals

Evaluates business strength using real-time price data from Yahoo Finance alongside live news context.

Agent 04

Bull Agent

Constructs the strongest possible case for buying. Argues from the data, not from bias. Passed to synthesis.

Agent 05

Bear Agent

Counters with the strongest case against. Identifies risks and challenges the bull agent's assumptions directly.

Agent 06

Synthesis

Weighs all five agents as a senior fund manager. Produces a final verdict with confidence rating and full reasoning.

Capabilities

What makes it different

Retrieval-Augmented Generation

Agent 01 retrieves real web data before any analysis begins. All downstream agents reason from current facts, not training knowledge alone.

Bull vs Bear debate

Two agents argue opposite sides of the trade before synthesis. This mirrors how real investment committees make decisions.

Live price data

Real-time price, daily change, and 30-day momentum pulled from Yahoo Finance and passed directly to the fundamentals agent.

Data visualizations

Radar profile, 30-day price chart, revenue growth model, and 52-week range — all generated dynamically per analysis.

Conversation memory

Ask follow-up questions after the report. The system passes full report context into every turn of the conversation.

Token transparency

Every session displays exact tokens consumed across all six agent calls — approximately 5,000–9,000 tokens per full analysis.
